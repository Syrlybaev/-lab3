Виртуальный деканат для ЛР №3 по дисциплине "Информационные системы и технологии".

Реализовано:
- идентификация пользователя по логину;
- аутентификация по постоянному многократно используемому паролю;
- дискреционная авторизация на базе Access Control List (ACL);
- ролевое управление интерфейсом приложения;
- роли: студент, преподаватель, администратор;
- разделы:
  студент -> оценки, расписание;
  преподаватель -> студенты, оценки, расписание;
  администратор -> студенты, преподаватели, оценки, расписание, управление пользователями.

Демо-вход:
- student_demo / stud2026
- teacher_demo / teach2026
- admin_demo / admin2026

Запуск:
python main.py
