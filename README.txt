Объединённая финальная версия проекта "Виртуальный деканат"

Что включено:
1. Сохранён функционал исходного приложения одногруппницы:
   - вход в систему;
   - просмотр расписания;
   - просмотр успеваемости;
   - меню по ролям.
2. Добавлен и встроен модуль "Преподаватели".
3. Архитектура разбита на файлы, чтобы проект было удобно вести командой.
4. Добавлены служебные экраны "Пользователи" и "Финансы".
5. Есть два демо-метода входа: пароль и токен.

Запуск:
python main.py
